# Trabjo-micros

FUNCIONALIDAD:

Tendrá dos modos de funcionamiento, dia y noche .

En el modo "noche", la alarma se activará al detectar luz,presencia o sonido.

En el modo "dia", la alarma se activará al detectar presencia o sonido.

Se detectará si es de dia o de noche con una entrada digital, la cual será 1 o 0, regulada por un temporizador.
Si se activa la alarma, tendrás 20seg para desactivarla, mientras suena la alarma. Si pasán los 20seg, se llamará a la policía para que vengan a por alcolea, cerrando las puertas y ventanas automáticamente.
